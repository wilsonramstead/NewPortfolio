The Wall

  • Description

    • |

  • Tech

A website where users can register an account and create posts to a public wall. Users can comment on other users posts, and visit their customized account page. Users have a network feature to list all users they are and are not connected to. Users can have private direct message conversations with other users, and create a private wall with a group of their connections.

Front End: Html, Css, Javascript, jQuery, Bootstrap

Back End: Asp.Net Core, Entity Framework

Database: MySQL
