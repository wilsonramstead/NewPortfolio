What is full stack developement?

A full stack web developer is a person who can develop both client and server software.

In Addition to mastering HTML and CSS, he/she also knows how to:

  • Program a browser (like using JavaScript, jQuery, Angular, or Vue).
  • Program a server (like using PHP, ASP, Python, or Node).
  • Program a database (like using SQL, SQLite, or MongoDB).

Reference: w3schools.com

See Portfolio for examples of this!

Full Stack Web Developement involves developing and designing the following three layers:

  • Front-End layer - browser (what the user can visually see on the screen).
  • Back-End layer - server (the backend server to handle all browser requests).
  • Database layer (to store user data + CRUD operations (create, read, update, delete) regarding accounts).
